Savor

Savor is a website that is designed to give the user lyrics to a song that they desire. In addition to this, Savor provides information about the artist as well as a potrait of the artist's album cover. At the bottom of the artist's description box there is a link to the artist's Apple Music page where the user can find more information about the given artist. The artist description, portrait, and link to Apple Music, was all retrieved using Apple Music's API. The lyrics were obtained by calling lyrics.ovh's API.

Weather Dashboard

The Weather Dashboard project is a website that is designed to show the 5-day weather forecast for a given city that the user is interested in. In addition to the 5-day forecast, the website also provides the current weather for the city of interest, such as the temperature in degrees fahrenheit, humidity, and wind speed. Lastly, the weather dashboard offers a search history function in which the user can view any previous city's that they have searched for in the past.

Password Generator

This Password Generator project allows the user to create a randomized password. The password length can be customized from 1-50 characters. The characters used in the password can be capitalized and consist of most characters available on the keyboard. The website also has a copy to clipboard feature, giving the user the option to copy their password and paste it in another location.

Eat-Da-Burger

Eat-Da-Burger is a restaurant app that lets users input the names of burgers they'd like to eat. When the user submits the name of the burger they would like to eat, it is placed on the left side of the page. Once the user clicks on the 'Devour it!' button, the burger will be devoured and moved to the right side of the page.
